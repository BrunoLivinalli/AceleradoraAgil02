# Galeria de Fotos 

Aplicação web desenvolvida com **HTML, CSS e JavaScript**, que exibe uma galeria de fotos em formato de grid e permite **pesquisa em tempo real** com base no nome das imagens.

---

## Tecnologias Utilizadas

- **HTML5** — Estrutura da página  
- **CSS3** — Estilização, Grid Layout, Flexbox e Responsividade  
- **JavaScript (ES6)** — Manipulação do DOM e eventos  
- **Font Awesome** — Ícones da interface  
- **LiveServer** - Expansão VSCode para servidor local

---

##  Como Usar a Aplicação


1. Abra a aplicação no VScode 
2. Adicione a extenção LiveServer
3. Inicie a extenção no canto inferior direito
4. Ao carregar a página, será exibida a aplicação automaticamente.

---

### Pesquisando Fotos

1. Localize o **campo de busca** na parte superior da página.
2. Clique no campo de texto e digite uma palavra-chave.
3. A filtragem acontece **em tempo real**, sem a necessidade de pressionar Enter.
4. Apenas as fotos cujo **nome contenha o texto digitado** serão exibidas.
5. Caso nenhuma foto corresponda à pesquisa, a seguinte mensagem será exibida:

   > *Nenhuma foto encontrada*

---

### Interagindo com as Imagens

- Passe o mouse sobre qualquer imagem para visualizar um **efeito de destaque**.
- O efeito visual ajuda a identificar qual imagem está sendo selecionada.
- Abaixo de cada imagem é exibido o **nome correspondente da foto**.
